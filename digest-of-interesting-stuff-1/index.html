Emacs

Gist.el

As an Emacs user I found this stuff very useful. I use GitHub Gist a lot. Before switching, starting to live in Emacs I had a deal with VSCode, Atom, PHP Storm as well and always first of all looked for that possibility of gist turned on. When you have a problems with code, optimizations, whatever related to programming, you gonna need "temporary paste codes thing", to describe your problem better with code in dev-community/chat and these can be: Pastebin, GitHub Gist, Glot.io, IdeOne, etc. I assume Pastebin is used a lot, and so is GitHub Gist. I consider gists more useful than pastebins, I don't know, I just like it, it's "quick temp code paster" thing I need.

I was a bit upset about that sort of thoughts which were used to tell me, that gists does not exist for Emacs, but fortunately it does exist. Now I'm using it much-much often. Features are incredibly useful. You can list all your gists within Emacs, edit them, either delete also or create new ones.
